# GT6testMOD-JAPANESEversion-
グランツーリスモ6に100個の新イベントを追加するMODです。このMODは、**GT6をクリアし、既存イベントだけでは満足できなくなってしまったプレイヤーのために**、入門クラスからスーパークラスまで合わせて**100個のイベント**を用意しました。**このMODを入れて新たなグランツーリスモワールドを始めよう！**

## 本MODで追加される新イベントの一覧
入門クラス

1.[エブリデーカップ](#エブリデーカップ)

2.[サニーカップ](#サニーカップ)

---

### 入門クラス
<a id="エブリデーカップ"></a>
#### 🏁 エブリデーカップ
新たなグランツーリスモワールドに足を踏み入れたばかりのプレイヤーを対象にしたレースイベントです。毎日グランツーリスモをプレイしているあなたには、サンデーカップではなくエブリデーカップがおすすめ！

##### レギュレーション
PP制限：350以下

タイヤ制限：スポーツハード以下

カート：禁止

##### 主なライバル車
ホンダ：フィット RS'10

ミニ：クーパー S '02

フォルクスワーゲン：ポロ GTI '01
